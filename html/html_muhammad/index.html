Sometimes you might want to disabe the users from right clicking a portion of your web page or the whole web page, this can be done using html only

=> <body oncontextmenu="return false"> </body>

Insert the oncontextmenu attribute to the body of your webpage to disable right click on the entire web page


=> <section oncontextmenu="return false"> </section>

This is use to disable right click on the this portion of the web page i.e the content in this section tag (right click is disabled)

For user who are on mobile devices, you can use the <input/> with a capture attribute to open the user's cameras and allow them to take a photo or video to upload to your website!

    
    

    <input type="file" id="user" capture="user" accept="image/*"/>  This will open the users camera on mobile devices it can be used to campture the user and environment 
           
    <input type="file" id="environment" capture="environment" accept="video/*"/> This will open the users camera on mobile devices it can be used to for the purpose of video only

The input tag with the "user" attribute and "image/*"

This will open the users camera on mobile devices it can be used to campture the user and environment

The input tag with the "environment" attribute and "video/*"

This will open the users camera on mobile devices it can be used to video the user environment only
